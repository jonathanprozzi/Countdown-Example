Countdown to [???]

This is a fun project where you can have the viewer specify an amount of time to count-down (within reason!) and then it's up to you to ... make ... something ... happen!
Some ideas are to have several elements appear, change the colors of the page, add images, etc. Basically, the requirement is to have some sort of *new* things happen once the timer hits 0.
